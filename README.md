# 🚗 Fleet Management Platform — Real-Time Java Telematics
A comprehensive Java Enterprise-grade vehicle telematics ingestion engine designed for corporate fleet logistics and driver safety optimization.

### 🛠️ Architecture Stack
* **Core Backend:** Java 17 / Java 21 LTS
* **Package Structure:** Standard Multi-Tier Enterprise Architecture
* **Protocols Handled:** OBD-II (SAE J1979), CAN Bus (ISO 11898), GPS/GNSS (NMEA 0183)
